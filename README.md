## 👨‍🔬 个人简介 (About Me)

我是左娅妮，现任中国计量科学研究院（NIM）时间频率计量科学研究所副研究员。2014年本科毕业于北京交通大学（专业第一），2020年博士毕业于清华大学物理系（师从王力军教授与张建伟副教授）。2020年至2023年于中国计量科学研究院从事博士后研究（合作导师：**李天初 院士**）。目前作为骨干成员就职于计量院基础创新团队——**喷泉与离子光钟团队**（负责人：房芳 副院长）。

我的主要研究方向为**时间频率计量与量子基准关键技术**。近年来聚焦于镱离子（¹⁷¹Yb⁺）光钟关键技术、镉离子（¹¹³Cd⁺）微波钟核心技术以及集成化超稳微波激光系统的研制。部分核心成果已成功实现转化应用，服务于航天二院203所、航天五院504所等星载钟及重大任务。在 *Nature Communications*, *Optics Letters*, *Applied Physics Letters* 等学术期刊发表多篇论文，并荣获中国计量测试学会科学技术进步一等奖、IEEE IFCS 最佳学生论文冠军奖等荣誉。

## 🔬 研究方向 (Research Interests)

* **高精度光频标与量子基准：** 镱离子（¹⁷¹Yb⁺）光钟实验装置搭建、钟跃迁激光系统性能提升与腔稳频技术。
* **时频信号处理与比对：** 钟信号的高效传递与低噪声处理、光钟系统间的高精度频率比对。
* **物理效应评估与抑制：** 离子光钟黑体辐射频移与光频移的不确定度评估与降低技术。
* **微波频标技术：** 镉离子微波钟协同冷却与稳定囚禁、超稳微波激光系统研制。
