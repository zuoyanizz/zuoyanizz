## 个人简况

左娅妮，女，博士，中国计量科学研究院时间频率计量科学研究所（国家时间频率计量中心）副研究员。2020年毕业于清华大学物理系，师从王力军教授与张建伟副教授，从事基于协同冷却镉离子微波钟的研究。2020年至2023年在中国计量科学研究院时频所从事博士后研究，2023年起聘为副研究员（事业编制）。

主持国家自然科学基金青年项目1项、国家市场监督管理总局科技计划项目1项及院基本科研业务费项目2项，作为骨干参与合肥国家实验室科技创新2030重大项目、科技部NQI国家重点研发计划等重要课题。主要科研贡献包括镱离子光钟关键技术攻关、镉离子微波钟核心技术突破以及集成化超稳微波激光系统研制，部分成果已实现转化应用，服务于航天院所重大任务。

在 *Nature Communications*、*Optics Letters*、*Applied Physics Letters*、*物理学报*、*计量学报* 等期刊发表SCI/EI论文多篇，申请多项发明专利。曾获中国计量测试学会科学技术进步奖（2025，排名3/10）、IEEE国际频率控制大会最佳学生论文竞赛组冠军奖（2018，大陆科研单位首次获奖）等荣誉。

## 研究方向

**1. 镱离子光钟关键技术**

主持国家自然科学基金青年项目，参与科技部NQI重点研发专项，开展腔稳频激光系统研制、离子阱搭建、频率不确定度建模与绝对频率测量等核心技术研究；推动腔体热噪声优化、激光消噪声技术成果转化，并参与国防科工局重点任务联合攻关；相关成果已应用于航天二院203所离子钟项目。

**2. 镉离子微波钟核心技术**

参与"镉离子微波钟技术"项目，负责协同冷却与离子稳定囚禁技术研究；技术成果已在203所和五院504所星载钟项目中应用；项目获中国计量测试学会科技进步奖，相关论文获IEEE IFCS最佳学生论文竞赛组冠军奖与中国时间频率年会优秀论文奖。

**3. 集成化超稳微波激光系统**

参与国防科工局"超稳微波"项目，联合北京大学等单位开展高相干双波长激光与电光梳分频研究，推动低噪声光学频率基准在微波领域的应用；相关成果发表于 *Nature Communications*（2025）。
